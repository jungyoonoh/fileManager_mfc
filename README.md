# fileManager_mfc
사진, 음악 파일등을 분류 및 정리해주는 솔루션

## 개발 배경 
여행 갔다오면 쌓이는 수많은 파일들을 효과적으로 정리할 수 없을까?<br>

### 목표
- 메타데이터를 바탕으로 폴더를 만들고, 하위 제목으로 사진 파일명을 일괄적으로 변경할 수 있도록 한다.<br>
- 사용자 기준에 따른 파일 일괄 정렬을 할 수 있도록 한다.
